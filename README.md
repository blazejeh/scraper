Portfolio Scraper - PHP Project
==============================

1. Uruchom XAMPP: Apache + MySQL
2. Stwórz bazę: scraper_portfolio
3. Wejdź: http://localhost/moj-scraper/

Funkcje:
- Dodawanie URL-i do kolejki
- Usuwanie pojedyńczego URL
- Usuwanie Wszystkich URL-i
- Scraping e-maili i telefonów
- Format telefonów: xxx xxx xxx
- Usuwanie duplikatów (UNIQUE + INSERT IGNORE)
- Zapis do MySQL (tabela kontakty)
- Panel wyników z tabelą
- Eksport do CSV (z separatorem ;)
- Czyszczenie bazy + reset ID
